　　　　　　 Customer journey

1. Login
GestボタンからだとSign upすることなく、ログインできます。
2. Home
全ユーザーの投稿一覧（公開設定の投稿のみ）
2. Home -Refined search
作品に付けられた製作方法を表すタグによる絞り込み検索が行えます。
2. Home -Menu
PCと一部のタブレットはヘッダーメニューから、スマートフォンと一部のタブレットはハンバーガーメニューからページ遷移します。
3. Favorite
お気に入り登録した投稿の一覧
3. Favorite -Refined search
作品に付けられた製作方法を表すタグによる絞り込み検索が行えます。
4. Private
あなたの投稿一覧
4. Private -Add
新規投稿　タグを追加できます。
4. Private -Edit
投稿を編集
5. Profile
ユーザー情報が表示されます。

Concept

「あなたが繋ぐ創作の輪」
課題
写真と文章がバラバラに保存される！
デザインが偏る！
新しい技術の知り方が分からない！
他の人と共有したい！
Myhandmadesを使ったら…
写真と文章が一緒に保存される！
あなたもみんなもデザインの幅が広がる！
あなたもみんなも新しい技術を知れる！
他のユーザーと共有できる！しかも投稿ごとに公開、非公開を選べる！
こだわり
制作方法をタグに
タグ（制作方法）をユーザーが追加できる
タグ（制作方法）ごとに投稿数をカウントする
公開時にusernameを出さない
苦労した点
iPhoneとWindowsでのUIの違い
お気に入り機能において、iPhoneではお気に入り登録前はcheckboxが消えず、お気に入り登録後はcheckboxが消える。
対応策：
checkboxの疑似クラスは使用せずに、divタグで☆と★を描き、お気に入り登録状況に連動させる。
編集機能を各作品ごとに独立させること
編集ボタンにidをつけると、onclickイベントにおいて、引数が作品IDではなく、編集ボタンのidになる。
対応策：
モーダルウィンドウをinputタグid属性とlabelタグfor属性を連動させて表示するのでなく、aタグのhref属性から表示させることで、id属性をつけない。
