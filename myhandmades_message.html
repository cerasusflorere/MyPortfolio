Outline this app

1. Login
GestボタンからだとSign upすることなく、ログインできます。
2. Home
全ユーザーの投稿一覧（公開設定の投稿のみ）
2. Home -Menu
PCと一部のタブレットはヘッダーメニューから、スマートフォンと一部のタブレットはハンバーガーメニューから
ページ遷移します。
3. Favorite
お気に入り登録した投稿の一覧
4. Private
あなたの投稿一覧
4. Private -Add
新規投稿　タグを追加できます。
4. Private -Edit
投稿を編集
5. Profile
ユーザー情報が表示されます。

Why I made this app?

私は手芸作品のデザインを決めるとき、Instagramやハンドメイド販売サイトを閲覧しています。
しかし、なかなか使用した素材や技法など得たい情報が載っていないのが実情です。
そこで、手芸作品に特化したSNSであれば自分の作品の管理ツールとしても使用でき、
それを共有することで、さらに創作の幅が広がるのでは、と考えました。
